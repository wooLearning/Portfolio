# Full-Custom-IC Design 교육 | Full-Custom-IC Design Training

## 1. 교육 정보 | Training Information
- 교육명 / Program: `Full-Custom-IC Design 교육`

## 2. 한국어 요약 | Korean Summary
Cadence Virtuoso와 Spectre를 활용해 기본 논리게이트와 4비트 전가산기/감산기를 설계했습니다.  
회로도 작성, 시뮬레이션, 레이아웃, DRC/LVS 검증까지 수행하며 full-custom 디지털 회로 설계 흐름을 실습했습니다.

## 3. English Summary
Using Cadence Virtuoso and Spectre, this training covered the design of basic logic gates and a 4-bit adder/subtractor.  
The full flow was practiced end-to-end, including schematic entry, simulation, layout, and DRC/LVS verification.

## 4. 구현 블록 | Implemented Blocks
### Logic Gates
- `NOT`, `2/3/4-NAND`, `2/3/4-NOR`, transmission-gate switch
- Schematic, simulation, layout, and PMOS width tuning

### Multiplexers
- `2x1`, `4x1` MUX
- Static-gate vs transmission-gate style comparison

### Adders
- Half adder, full adder, 4-bit adder/subtractor
- Verified by schematic/simulation/layout outputs
