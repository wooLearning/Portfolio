## 3. English Summary
This training covered the full PCB workflow from simulation to hardware validation.  
After practicing with Capture and PCB Editor, the designed board was soldered and measured with an oscilloscope for final verification.

## 4. 문서 구성 | Document Set
- Portfolio 1: PSpice simulation practice
- Portfolio 2: Capture schematic workflow
- Portfolio 3: PCB setup, constraints, routing, artwork export
- Final report: elevator PCB implementation and validation

## 5. 수행 내용 | Work Performed
### Simulation and Schematic
- Signal source and analysis mode setup
- Module/port configuration and hierarchical schematic
- ERC/DRC rule checks

### PCB Layout
- Netlist import and library mapping
- Outline/keepin/cross-section setup
- 4-layer routing (`TOP/GND/VCC/BOTTOM`)
- Padstack/spacing/power constraints and DRC cleanup

### Final Elevator PCB
- Oscillator/divider/encoder/latch/counter/comparator/decoder based board integration
- Practical soldering and TP-based waveform measurement

## 6. 기술 스택 | Tech Stack
- Simulation: `PSpice`
- Schematic: `OrCAD Capture`
- PCB: `PCB Editor`
- Validation: `DRC/ERC`, oscilloscope measurement
