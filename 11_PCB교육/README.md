# PCB Circuit Design & PSpice
> PCB Design Implementation Projects

## 📅 Project Info
- **Period**: 2025.07 (60 Hours)
- **Org**: NinePlusIT (나인플러스IT)
- **Stack**: `OrCAD Capture` `PSpice` `PCB Editor`

## 📝 Summary
회로 시뮬레이션부터 PCB 제작, 납땜 및 검증까지 **하드웨어 구현의 전체 흐름**을 경험했습니다.  
PSpice를 이용한 회로 해석, OrCAD Capture를 이용한 도면 작성, PCB Editor를 이용한 4-layer 라우팅을 수행하고, 최종적으로 엘리베이터 제어 보드를 제작하여 오실로스코프로 동작을 확인했습니다.

## 💡 Key Activities
- **Simulation**: Signal analysis & hierarchical block verification.
- **PCB Layout**: 4-Layer Routing (Top-GND-VCC-Bottom), Design Rule Check (DRC).
- **Fabrication**: Soldering components and waveform measurement (Oscilloscope).

## 📂 Artifacts
- Portfolio 1~3 (Simulation, Schematic, PCB)
- Final Report (Elevator PCB Implementation)
