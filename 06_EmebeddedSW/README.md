# 임베디드 소프트웨어 (ATtiny to STM32)
> Embedded Software Project Portfolio

## 📅 Project Info
- **Period**: 2024.12.23 ~ 2025.01.24
- **Platform**: `ATtiny2313A` `STM32F401CC`
- **Stack**: `C` `GPIO` `Timer` `SPI` `UART`

## 📝 Summary
AVR(ATtiny)부터 ARM(STM32)까지 다양한 MCU를 활용한 **임베디드 제어 및 응용 프로젝트**입니다.  
기초적인 타이머/인터럽트 제어부터 시작하여, STM32 기반의 그래픽 렌더링(ILI9341)과 터치 게임 구현까지 단계적으로 시스템 통합 역량을 개발했습니다.

## 💡 Project Tracks
1.  **ATtiny2313A Timer**: Timer Start/Reset 로직 및 I/O 제어.
2.  **ATtiny4313 Integrated**: UART 통신, PWM 밝기 제어, 인터럽트 처리 통합.
3.  **STM32 Rendering**: ILI9341 디스플레이 DMA 그래픽 렌더링, Encoder 연동.
4.  **STM32 Touch Game**: SPI 기반 터치스크린 카드 매칭 게임 구현.
