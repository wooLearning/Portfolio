# Full-Custom-IC Design 교육
> Full-Custom-IC Design Training

## 📅 Project Info
- **Period**: 2024.07 ~ 2024.08 (60 Hours)
- **Org**: NinePlusIT (나인플러스IT)
- **Stack**: `Cadence Virtuoso` `Spectre` `Assura` `GPDK090`

## 📝 Summary
Cadence Virtuoso와 Spectre를 활용하여 **Full-custom 디지털 회로 설계** 전 과정을 실습했습니다.  
논리 게이트(NAND/NOR/NOT)부터 4-bit Adder/Subtractor까지 Schematic 설계, Simulation, Layout, DRC/LVS 검증을 수행했습니다.

## 💡 Implementation
- **Logic Gates**: NOT, NAND, NOR, Transmission Gate 설계 및 PMOS/NMOS 사이징.
- **Multiplexers**: 2x1, 4x1 MUX (Static vs TG 방식 비교).
- **Arithmetic**: Half Adder, Full Adder, 4-bit Adder/Subtractor 레이아웃 구현.
